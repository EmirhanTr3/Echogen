# Echogen

Echogen is a powerful and versatile utility plugin for modern Minecraft servers, designed to enhance player experience and provide administrators with extensive control over their server's functionality.

## ✨ Features

Echogen is packed with features, all highly configurable to suit your server's needs.

### 💬 Advanced Chat & Formatting

*   **Custom Chat Format:** Take full control over how chat messages appear. Using [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/), you can create complex and beautiful chat formats.
*   **PlaceholderAPI & LuckPerms Integration:** Seamlessly display player prefixes, suffixes, and any other placeholder in chat.
*   **Item Display:** Allow players to show off their items in chat with `[item]`
*   **Regex-Based Chat Filter:** Keep your chat clean with a powerful and configurable filter that can censor or block messages based on regular expressions.
*   **Chat Moderation:** Easily manage chat with built-in commands to **mute**, **clear**, or enable **slowmode**.
*   **Custom Join & Quit Messages:** Configure unique messages for when players join or leave the server.

### 🎨 Prefix Manager

*   **Custom Prefixes:** Define a list of prefixes that players can choose from. Each prefix can have a unique style and description.
*   **Prefix GUI:** Players can easily browse and select their desired prefix through an intuitive graphical interface.

### 👻 Vanish System

*   A complete vanish system that allows staff members to become invisible to other players, perfect for moderation and observation.

### 📊 Scoreboard

*   **Custom Scoreboard:** Create a custom scoreboard for your players with a configurable title and lines.
*   **MiniMessage & Placeholders:** Fully customize the scoreboard's appearance and display dynamic information using [MiniMessage](https://docs.papermc.io/adventure/minimessage/format/) and [PlaceholderAPI](https://modrinth.com/plugin/placeholderapi).
*   **Support for [Skript](https://modrinth.com/plugin/skript)**: Placeholders to display value of Skript variables and output of Skript functions.

### 📈 Performance Monitoring

*   **TPS Bar:** Monitor server performance (TPS, MSPT, Ping) via a toggleable BossBar (`/tpsbar`).
*   **RAM Bar:** Keep track of memory usage (Used, Max) via a toggleable BossBar (`/rambar`).

### 🛠️ Utility Commands

Echogen comes with a suite of essential commands for server administrators and players, including:

*   `/fly`: Toggle flight for yourself or other players.
*   `/sudo`: Force a player to run a command or send a chat message.
*   `/smite`: Bring the thunder down on a player.
*   `/enderchest`: Open your own or another player's ender chest.
*   `/chat`: Manage chat settings (clear, mute, slowmode).
*   `/tpsbar` & `/rambar`: Toggle performance monitoring bars.
*   And many more!

## ⚙️ Configuration

Echogen is built to be highly configurable. Almost every feature can be enabled, disabled, or modified in the `config.yml` file. Reload the configuration in-game to apply changes instantly without needing to restart your server.

---

This plugin leverages modern PaperMC APIs for performance and stability. It's the perfect all-in-one solution for server owners looking to add essential features without the bloat of multiple smaller plugins.
